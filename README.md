# Read and Write .txt Files

Hello, this is a simple project to learn basics of reading and writing .txt files.

# Why?

This project is for beginner who tryna learn. It it just txt right now. You can do that same thing with json with java json libraries.

# How to use?

This repository has 2 files contain code. When you download this repository you can use it like that.

## How to Write?

First run the Write.java. when you run it just enter the thing you want to save then press enter.

## How to Read?

You have to run the Read.java if you want to see the thing you wrote before.
